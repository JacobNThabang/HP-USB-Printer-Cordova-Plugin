# HP-USB-Printer-Cordova-Plugin

An Android Cordova USB Printing Plugin for the: 
```HP Engage One Prime - Receipt printer - thermal paper - Roll (5.8 cm) - 203 dpi - up to 90 mm/sec``` 
using the HP Point of Sale SDK for Android.  

Android Device that this is currently being developed for: ```HP Engage One Prime Receipt Thermal Printer```
